wkx-sharp [![Build status](https://ci.appveyor.com/api/projects/status/a7501564auswvihn?svg=true)](https://ci.appveyor.com/project/cschwarz/wkx-sharp)
========

A WKT/WKB/EWKT/EWKB parser and serializer with support for

- Point
- LineString
- Polygon
- MultiPoint
- MultiLineString
- MultiPolygon
- GeometryCollection
